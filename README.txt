# Championnat Basket Loisir 35

Le comité 35 du Basket a lancé un nouveau championnat loisir lors de la saison 2024-2025.
Cette petite application permet de référencer les clubs qui participent à cette compétition.

Plus d'information à cette addresse : `https://illeetvilainebasketball.org/centre-de-ressources/faq-categorie/23/basket-loisirs`
